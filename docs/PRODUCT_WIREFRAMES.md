Wireframe Textual – KindX
1. Tela de Boas-Vindas
- Logo KindX
- Slogan: “Empowering Communities Through Help”
- Botões: Login | Sign Up
- Opções de login: Continue with Google | Continue with Apple
2. Cadastro / Login
- Campos: Email / Senha
- Botão de login social (Google/Apple)
- Link: Forgot password?
3. Tela Inicial (Home)
- Barra superior: logo + ícone de notificações 🔔
- Opções principais: Ask for Help | Offer Help | Community Feed
- Navegação inferior: Home | Requests | Offers | Profile
4. Pedir Ajuda (Ask for Help)
- Campos: Título, Descrição, Categoria, Localização (GPS ou manual)
- Botão: Post Request
5. Oferecer Ajuda (Offer Help)
- Campos: Tipo de ajuda, Disponibilidade (data/horário), Localização
- Botão: Publish Offer
6. Lista de Pedidos Próximos
- Filtro por: categoria | distância | data
- Cards: Nome do usuário, Pedido, Localização, Botão Offer Help
7. Chat / Mensagens
- Lista de conversas
- Tela individual: Nome do usuário, Mensagens em bolha, Botão Confirmar ajuda
8. Perfil do Usuário
- Foto + Nome
- Bio / Descrição
- Estatísticas: Ajuda recebida | Ajuda oferecida
- Botões: Edit Profile | Settings
9. Histórico
- Pedidos concluídos
- Ajuda oferecida
- Avaliações (feedback, estrelas)
10. Notificações
- Alertas de atividades recentes
- Exemplo: João ofereceu ajuda | Maria pediu ajuda perto de você
11. Configurações
- Idioma (EN/PT)
- Privacidade
- Logout
